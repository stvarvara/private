Outils

Dans ce projet, j'ai contribué à la création d'une base de données pour le festival de musique Rencontres Trans Musicales en travaillant avec mon camarade de classe.

Travail réalisé

  • Traitement des fichiers de données à l'aide de scripts PHP pour la génération de PDF destinés aux équipes commerciales régionales.
  • Création d'une maquette HTML en tant que modèle pour les fichiers PDF générés.
  • Conversion des photos en PNG en nuances de gris en utilisant un conteneur Docker pour assurer la cohérence et la qualité des images.
  • Génération de QR-codes pour les intégrer dans les fichiers PDF à l'aide d'un conteneur Docker pour garantir une performance optimale.
  • Documentation détaillée pour faciliter l'utilisation et la maintenance de l'outil développé.
