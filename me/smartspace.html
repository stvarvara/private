Outils

J'ai conçu SmartSpace, un site web pour une organisation spécialisée dans la vente de maisons. Ce projet m'a permis de mettre en pratique mes compétences en JavaScript et de développer mes connaissances en matière d'adaptation de sites web pour répondre aux besoins spécifiques de l'organisation. Le site web offre une expérience utilisateur fluide et intuitive pour aider les clients à trouver leur maison idéale.

Travail réalisé

  • Analyse de maquette pour comprendre les fonctionnalités et les éléments visuels attendus dans le site web.
  • Codage des pages en utilisant des langages de balisage tels que HTML et CSS pour la structure et la présentation du contenu.
  • Ajout des fonctionnalités interactives au site web en utilisant le langage de programmation JavaScript.
  • Tests et vérifications pour garantir la qualité et la fonctionnalité du site web.
  • Adaptation pour les différentes tailles d'écran et résolutions, afin que les utilisateurs puissent facilement naviguer sur le site web à partir de différents appareils.

Soft skills

  • Autonomie
  • Flexibilité
  • Capacité à se concentrer sur les détails
